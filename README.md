# Design and Analysis of Algorithms

## Project Title
Sorting Algorithm Performance Analysis

## Description
This project analyzes the performance of different sorting algorithms by measuring their execution time on a given dataset.

The project demonstrates the concepts of:
- Algorithm Design
- Algorithm Analysis
- Time Complexity
- Performance Comparison

## Algorithms Implemented

### 1. Bubble Sort
- Technique: Brute Force
- Time Complexity:
  - Best Case: O(n)
  - Average Case: O(n²)
  - Worst Case: O(n²)

### 2. Selection Sort
- Technique: Comparison Based Sorting
- Time Complexity:
  - Best Case: O(n²)
  - Average Case: O(n²)
  - Worst Case: O(n²)

## Dataset
The dataset contains random integer values that are used for sorting and performance testing.
